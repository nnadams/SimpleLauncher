-------------------------------------------------------------------------------
                                 Release Notes                                 
-------------------------------------------------------------------------------
Enid, by SL0, allows the user to import files and then launch them with a      
designated program. It allows the user to create butttons and place them in    
different places in the program. Enid also provides a "night" mode where the   
background changes to a dark grey. The program is particularly useful in a     
situation where the user needs to launch multiple videos in a certain order.   
                                                                               
No install needed, just download and run.                                      

-------------------------------------------------------------------------------
                                 Install Notes                                 
-------------------------------------------------------------------------------
1) Download Enid                                                               
2) Run!                                                                        
3) When saving, the program saves a .dat that must be in the same directory,   
   if using that save, and it will be automatically opened.
